# 芯阳服务器单点登录最新方案

详细操作步骤总结

  1. 修改 Nginx 站点配置


# 将 server_name 从 192.168.8.185 改为 10.1.120.36

sudo sed -i 's/server_name 192.168.8.185;/server_name 10.1.120.36;/' /etc/nginx/conf.d/openwebui-sso.conf  
2\. 创建 .env 文件（含 SSO 参数）  
文件路径：/home/xinyang/.env  
在原有环境变量基础上，新增以下 SSO 参数：  
WEBUI_SECRET_KEY=162f1f6100312f528a154ffab946b8e41b41fdb54f7e5f4463234ab76e062257  
WEBUI_AUTH_TRUST_HEADERS=true  
WEBUI_AUTH_TRUSTED_EMAIL_HEADER=X-Internal-User-Email  
WEBUI_AUTH_TRUSTED_NAME_HEADER=X-Internal-User-Name  
3\. 停止并删除旧容器  
docker stop open-webui-0525 && docker rm open-webui-0525  
4\. 启动新容器（端口 8081，开启 SSO 认证）  
docker run -d -p 8081:8080 --add-host=host.docker.internal:host-gateway -v /var/lib/docker/volumes/openwebui-data-082/_data:/app/backend/data --env-file /home/xinyang/.env --name open-webui-0525 --restart always 223740cda2f76accc936eeb0e1349e2c37f09137ec7f76df79bc81e50025fa8f  
5\. 测试 Nginx 配置并重载  
sudo nginx -t # 测试配置语法  
sudo nginx -s reload # 重载使生效  
6\. 验证

# 验证容器运行

docker ps --filter name=open-webui-0525

# 验证 Nginx 反向代理正常

curl -o /dev/null -w "%{http_code}" [http://10.1.120.36:3001/](http://10.1.120.36:3001/)

# 验证登录页面可访问

curl [http://10.1.120.36:3001/login.html](http://10.1.120.36:3001/login.html)

# 验证 Cookie SSO 流程

curl --cookie "X_User_Email=admin@test.com; X_User_Name=admin" [http://10.1.120.36:3001/](http://10.1.120.36:3001/)  
最终架构  
用户访问  
[http://10.1.120.36:3001/login.html?user_email=xxx](http://10.1.120.36:3001/login.html?user_email=xxx)  
│  
▼  
Nginx (端口 3001)  
│ 从 Cookie 提取用户信息  
│ 转发为 X-Internal-User-Email / X-Internal-User-Name 请求头  
▼  
Open WebUI 容器 (127.0.0.1:8081)  
│ Trusted Header 认证自动登录  
▼  
使用中  
SSO 访问方式

  * 带邮箱登录： [http://10.1.120.36:3001/login.html?user_email=user@company.com](http://10.1.120.36:3001/login.html?user_email=user@company.com)
  * 指定用户名： [http://10.1.120.36:3001/login.html?user_email=user@company.com&user_name=张三](http://10.1.120.36:3001/login.html?user_email=user@company.com&user_name=张三)  
后续用户管理
  * 第一个 SSO 登录的用户自动成为管理员
  * 其他用户需管理员在「管理面板 → 用户」中手动创建并激活状态为「已激活」
