# 嘉立创会议纪要

本次会议主要介绍了嘉立创EDA的AI功能、插件开发方式，以及其一站式产业互联制造平台。

---

## 一、嘉立创EDA的AI功能与插件开发

### 1. AI功能介绍

会议展示了嘉立创EDA的多项AI能力：

- **在线批注评审**：支持实时在线批注和评审
- **AI助手**：集成了Kimi等AI助手
- **AI智能建库**：自动识别和创建元器件库
- **文生原理图**：通过自然语言生成原理图
- **AI辅助布局与布线**：智能辅助PCB布局布线

### 2. 插件扩展机制

- **JavaScript API**：提供完整的JavaScript API接口供开发者使用
- **扩展广场**：官方插件发布和下载平台
- **Cloud Code**：通过AI Agent工具，支持通过自然语言描述从0到1构建和调试插件，大幅降低开发门槛

---

## 二、嘉立创一站式产业互联制造平台

### 1. 产业链介绍

嘉立创形成了从设计到实物的完整闭环：

**EDA设计** → **PCB/PCBA制造** → **元器件采购（立创商城）** → **机械加工（3D打印、CNC、钣金等）**

### 2. 核心优势

| 优势 | 说明 |
|------|------|
| **自研工业软件** | EDA、MES、ERP等核心软件均为自主研发，保障数据安全与流程透明 |
| **高效协同** | 设计软件与制造流程打通，支持在线协同、版本管理和中央库管理 |
| **透明化服务** | 生产交期、价格等信息对客户公开透明，采用"先付款后生产"模式 |
| **强大产能** | 拥有600余台贴片机和300余条生产线 |
| **黑灯工厂** | 珠海仓库已实现自动取料与送料的全自动化 |

## 三、会议问答与讨论

### 1. 测试与成本

| 议题 | 结论 |
|------|------|
| **FCT测试架成本** | 高达数万元，打样阶段采用无需治具的飞针测试更经济，且能覆盖通断与焊接效果 |

### 2. 交期与产能

| 议题 | 结论 |
|------|------|
| **交期较长原因** | 严格的流程与物料管控；暂未布局海外基地，短期内重心仍会死磕国内供应链 |
| **产能优化** | 持续优化产能以提升交期与品质 |

### 3. 物料与库存

| 议题 | 结论 |
|------|------|
| **基础库** | 基于全平台销售数据筛选出的高频常用件，即全平台大数据统计的常用物料 |
| **基础库扩容** | 受限于单台设备100多种物料上限，盲目扩充会导致效率骤降，扩容概率极小，建议客户转向免物料费的优选库 |
| **基础库缺货** | 部分物料在市面已绝迹，官方正大力引导客户提前备货私有库 |
| **存储芯片缺货** | 价格暴涨，建议客户提前备料以应对缺货风险 |
| **电容缺货** | 缺货预警从电阻延伸至电容 |

### 4. BOM表与贴片配置

| 议题 | 结论 |
|------|------|
| **BOM表编辑** | 目的是让用户自助操作 |
| **贴片配置规则** | 邮寄物料时封装一致即可，代放则需分单 |
| **删除BOM中单个器件** | 可在同格内单独删除 |

### 5. 器件与贴片

| 议题 | 结论 |
|------|------|
| **贴片机取料** | 中间取料最快（以"桌子"作比） |
| **DDR贴片数量** | 若有疑问，建议自查"为何不贴三片"的需求 |
| **同器件不同物料** | 建议将同器件不同位号改为空贴以节省成本；若物料不同，可通过分格贴装实现 |
| **3D视图操作** | 器件删除逻辑为"左控右"，即左侧列表删除后右侧3D视图才会同步消失，无法从3D端反向操作 |

### 6. 费用与检测

| 议题 | 结论 |
|------|------|
| **焊点费优惠** | 嘉立创EDA专属优惠 |
| **降低换料费** | 建议通过购买整盘常用物料来大幅降低 |
| **X-Ray检测** | 用户现可自主决定BGA板子的抽检数量，按需付费 |
| **放弃X-Ray** | 若客户放弃X-ray检测，嘉立创仍可接受维修，但需客户自行承担交期延误风险 |

### 7. 自动布局布线

| 议题 | 结论 |
|------|------|
| **当前状态** | 仍有待提升 |
| **AI推进** | 团队正基于AI技术推进草图半自动布局布线 |

### 8. 线下活动

| 议题 | 结论 |
|------|------|
| **问卷填写** | 征集下一站线下活动的举办地，排除刚办过活动的东莞 |
| **深圳活动** | 下半年将在深圳总部举办大型活动，通过公众号获取资讯 |
| **苏州活动** | 若当地需求强烈便会安排前往 |
| **企业研修班** | 针对企业研发需求定制，为期7天在珠海举办，旨在统一研发规范与协作方式 |
