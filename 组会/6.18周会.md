## ✨ 一、关键进展

### 🔌 芯阳知识库 MCP Server 开发完成

> 芯阳公司内部知识库 MCP 服务器，封装内部搜索 API，兼容 **Codex** 和 **OpenCode** 两大 AI 编码助手。

| 📌 属性       | 说明                                                |
| :---------- | :------------------------------------------------ |
| 🏠 **项目地址** | https://github.com/chengyangliu-lcy/xinyangkb_mcp |
| 📦 **版本**   | v1.0.0                                            |
| 🛠️ **技术栈** | Node.js + TypeScript                              |

#### 🎯 功能一览

| 工具 | 描述 |
|:---|:---|
| 🔍 `knowledge_base_search` | 搜索芯阳内部产品参数、技术方案、制度、流程、项目和内部文档 |

#### 🏗️ 架构设计

```
┌─────────────────┐
│   MCP Server    │  ← src/index.ts（核心）
├─────────────────┤
│  知识库搜索 API │  ← knowledge_base_search Tool
├─────────────────┤
│   Codex/OpenCode│ ← AI 助手调用
└─────────────────┘
```

#### 📦 安装方式

**一键安装（推荐）：**

```bash
node scripts/install.cjs --api-url http://YOUR_HOST:5010/search/dify
```

| 平台 | 安装脚本 |
|:---|:---|
| 🐧 Linux / Mac | `./scripts/install.sh` |
| 🪟 Windows | `.\scripts\install.ps1` |

#### ✨ 支持的安装模式

| 模式            | 说明                        |
| :------------ | :------------------------ |
| `direct` (默认) | 直接注册到 Codex MCP           |
| `plugin`      | 通过 Codex Marketplace 插件安装 |
| `both`        | 同时安装两种模式                  |

#### 📖 行为规范

- 通过 `skills/xinyang-assistant/SKILL.md` 定义芯阳大模型行为规范
- 返回结果中的内部文档路径会转换为可引用 URL
- 支持环境变量和配置文件双重配置
