## ✨ 一、关键进展

### 1.1 🔌 LCSC MCP Server 开发完成

> 嘉立创(LCSC/JLCPCB)电子元器件查询 MCP Server。让 AI 助手能够搜索元器件、查看技术规格、获取数据手册。兼容 **Claude Code**、**OpenAI Codex CLI**、**OpenCode** 三个 AI 编码助手。

| 📌 属性       | 说明                                                                 |
| :---------- | :----------------------------------------------------------------- |
| 🏠 **项目地址** | [GitHub](https://github.com/chengyangliu-lcy/lcsc-mcp-server-node) |
| 📦 **版本**   | v1.0.0                                                             |
| 🛠️ **技术栈** | Node.js + TypeScript                                               |

#### 🎯 功能一览

| 工具 | 描述 |
|:---|:---|
| 🔍 `lcsc_search` | 按关键词搜索 LCSC 元器件库 |
| 📋 `lcsc_detail` | 按 C 编号获取元器件详细规格 |
| 📄 `lcsc_datasheet` | 获取元器件数据手册 PDF 链接 |

---

#### ⚙️ 安装指南

> **前置条件：** Node.js >= 18

**1. 全局安装**
```bash
cd lcsc-mcp-server-node
npm install
npm run build
npm install -g .
```

**2. 注册到 AI 工具**

| AI 工具 | 注册命令 |
|:---|:---|
| **Claude Code** | `claude mcp add --scope user lcsc -- lcsc-mcp` |
| **Codex CLI** | `codex mcp add lcsc -- lcsc-mcp` |
| **OpenCode** | 手动配置 `~/.config/opencode/opencode.json` |

> ⚠️ **注意：** Codex CLI 请写入 `~/.codex/config.toml`，新版不读取 `config.json`

**3. 验证安装**
```bash
codex mcp list
```

---

#### 🚀 一键安装脚本

| 平台 | Claude Code | Codex CLI | OpenCode |
|:---|:---|:---|:---|
| 🐧 Linux / Mac | `bash install_claude.sh` | `bash install_codex.sh` | `bash install_opencode.sh` |
| 🪟 Windows | `install_claude.bat` | `install_codex.bat` | `install_opencode.bat` |

### 1.2 📚 芯阳知识库 MCP 工具

> 芯阳公司内部知识库 MCP 服务器，封装内部搜索 API，供 **Codex** 和 **OpenCode** 调用。

#### 🏗️ 架构设计

```
┌─────────────────┐
│   MCP Server    │  ← src/index.ts（核心）
├─────────────────┤
│  知识库搜索 API │  ← knowledge_base_search Tool
├─────────────────┤
│   Codex/OpenCode│ ← AI 助手调用
└─────────────────┘
```

#### 🛠️ 技术选型

| 类别 | 选择 |
|:---|:---|
| 🌐 **语言** | Node.js + TypeScript (JS 69.3% / TS 27.3%) |
| 🔗 **协议** | MCP (Model Context Protocol) |
| 📋 **环境** | Node.js >= 18 |

#### ⚙️ 环境变量配置

| 变量 | 说明 | 示例 |
|:---|:---|:---|
| `SEARCH_API_URL` | 内部搜索 API 地址 | `http://10.24.116.22:5010/search/dify` |
| `API_SERVER_BASE_URL` | API 基础地址（拼接文档 URL） | - |
| `no_proxy` | 不走代理的内部地址列表 | - |

#### 📦 安装方式

- **一键安装：** `npm install` → `node scripts/install.cjs`（自动检测并配置）
- **手动安装：** 分别通过 Codex CLI 和 OpenCode 配置文件注册

#### 📖 行为规范

- 通过 `skills/xinyang-assistant/SKILL.md` 定义芯阳大模型行为规范
- AI 助手自动调用知识库搜索，按规范格式化回答
